setLogging

fun setLogging(level: LogLevel, listener: LogListener?): Unit

Enable/Disable Logging in the SFMCSdk.

Parameters

level - the LogLevel of the output. Setting this to LogLevel.NONE will disable logs.

listener - the LogListener to send the logs to. The SDK contains a default LogCat implementation as AndroidLogger. Setting this to null will disable logs.
