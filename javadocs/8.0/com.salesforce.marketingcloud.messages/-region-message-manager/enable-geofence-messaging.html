enableGeofenceMessaging

@RequiresPermission(["android.permission.ACCESS_FINE_LOCATION", "android.permission.ACCESS_BACKGROUND_LOCATION"]) abstract fun enableGeofenceMessaging(): Boolean

Enables geofence messaging. This will only succeed if:

  • com.salesforce.marketingcloud.MarketingCloudConfig.Builder#setGeofencingEnabled(boolean) (boolean)} was set to true
  • The device has GPS hardware
  • The play-services-location dependency was included in your build
  • The Manifest.permission#ACCESS_FINE_LOCATION permission has been granted.
  • The Manifest.permission#ACCESS_BACKGROUND_LOCATION permission has been granted on Android Q+.
  • The device has the supported play services version.

Return
Boolean: True if geofence messaging was successfully enabled
