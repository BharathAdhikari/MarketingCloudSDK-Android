RegionTransitionEventListener

Interface to use to listen for region(Geofence/Proximity) transition(enter/exit) events. The in the provided Region will always be empty when these methods are called. If there is a message attached to the region transition that should be shown as a notification then it will be displayed shortly after this interface is notified of the event.

enableGeofenceMessaging

Enables geofence messaging. This will only succeed if:

  • com.salesforce.marketingcloud.MarketingCloudConfig.Builder#setGeofencingEnabled(boolean) (boolean)} was set to true
  • The device has GPS hardware
  • The play-services-location dependency was included in your build
  • The Manifest.permission#ACCESS_FINE_LOCATION permission has been granted.
  • The Manifest.permission#ACCESS_BACKGROUND_LOCATION permission has been granted on Android Q+.
  • The device has the supported play services version.

abstract fun enableGeofenceMessaging(): Boolean

enableProximityMessaging

Enables proximity messaging. This will only succeed if:

  • com.salesforce.marketingcloud.MarketingCloudConfig.Builder#setProximityEnabled(boolean) was set to true
  • The AltBeacon dependency was included in your build.
  • The device has the correct hardware and API level (18)
  • The play-services-location dependency was included in your build
  • The Manifest.permission#ACCESS_FINE_LOCATION permission has been granted.
  • The Manifest.permission#ACCESS_BACKGROUND_LOCATION permission has been granted on Android Q+.
  • The device has the supported play services version.
